UC Campus Master Plan is an interactive 3D visualisation of the University of Canberra campus, designed to communicate current and future development plans. The tool allows users to explore the campus spatially and toggle between different planning scenarios.

Technical Highlights:

  • Built with Three.js, using optimised architectural models that were decimated prior to integration to maintain performance
  • Implemented layer-based visualisation by grouping meshes, allowing users to toggle different development plans on and off
  • Created a data-driven node system using JSON to display contextual information about buildings and development areas
  • Delivered the project under a tight multi-week deadline, focusing on clarity, performance, and reliability
