Tracksville Trivia is an interactive trivia experience for children, created as part of Rail Safety Week. The experience presents trivia questions within a playful 3D isometric environment, encouraging exploration and repeat engagement.

Technical Highlights:

  • Built using Three.js, creating a 3D isometric scene populated with interactive nodes representing trivia locations
  • Implemented a data-driven node system powered by JSON, allowing trivia locations and states to be managed independently of scene logic
  • Integrated with a custom trivia API, consuming question data and tracking progress as users moved through the experience
  • Managed scene state, visually updating nodes once questions had been answered
