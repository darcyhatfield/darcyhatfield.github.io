The Business Package Website is a reusable WordPress theme designed to be sold and deployed across multiple client sites. The goal was to give clients a powerful but approachable way to build pages without needing technical expertise.

Technical Highlights:

  • Architected as a Gutenberg-native theme using ACF Blocks for modular, reusable content sections
  • Built a highly flexible layout system supporting extensive configuration of typography, colour palettes, spacing, and layout variants
  • Implemented a theme options panel to manage global settings such as fonts, sizes, and brand colours
  • Designed for scalability and reuse, allowing the same theme to be deployed across multiple clients with minimal customisation
