Seeing Machines is a corporate WordPress website built to reflect the client's cutting-edge, technology-driven brand. A key requirement was a distinctive, “futuristic” homepage feature that visually reinforced the company's work in advanced machine vision and safety systems.

Technical Highlights:

  • Built as a custom WordPress theme using ACF to manage structured content throughout the site
  • Developed a Three.js hero banner, loading a 3D model with a custom shader that tracks mouse or gyroscopic movement
  • Balanced visual complexity with performance considerations to ensure smooth behaviour across desktop and mobile browsers
