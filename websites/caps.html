Caps Go Big is a website for a local basketball team, designed to make it easy for club staff to manage fixtures, ladders, and updates without technical knowledge

Technical Highlights:

  • Built with WordPress, using custom post types and ACF to structure fixtures and ladder data
  • Designed admin interfaces with non-technical users in mind, making manual data entry simple and error-resistant
  • Kept the implementation intentionally lightweight, prioritising clarity, ease of use, and maintainability
